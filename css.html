What is CSS?
  1. CSS stands for Cascading Style Sheets
  2. Css describes how the HTML elements are to be displayed on the screen,paper on in any other media
  3. CSS is not necessary CSS saves a lot of work.It can control the layout of the multiple web pages in all at once.
  4. external style sheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside the Html element
  • Internal - by using the style element inside the head section
  • External - by using a link element to link an external css file
  • The most common way to add CSS, is to keep the styles in external CSS files. However,in this tutorial we will use inline and internal styles, because it is easier to demonstrate and easier for you to try it yourself.

While comparing CSS class vs ID,the difference that the css class applies a style to multiply elements. ID on the other hand,applies the style to one unique element.ID is also
special in that you can use a special URL, to link directly to an element and its used by Javascript.

In CSS, Selectors are used to target a specific element on a web page.Once the element has been targeted,a style or set of styles can be applied to the
element.

There are wide range of selectors available, Two of the most commonly used are classes and ID. Both are used to target element in which style should be applied.
